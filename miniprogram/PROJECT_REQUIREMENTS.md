# 情侣空间微信小程序 - 交互需求文档

## 项目概述
情侣空间是一款专为情侣设计的微信小程序，帮助情侣记录美好时光、共享待办事项、上传照片并管理纪念日。

## 用户角色
1. **单身用户** - 未登录或未加入情侣空间的用户
2. **情侣用户** - 已登录并加入情侣空间的用户
   - 空间创建者
   - 空间加入者

## 核心功能模块

### 1. 首页 (Home)
**目标**: 提供情侣日常互动的核心功能入口

**交互需求**:
- 显示情侣状态信息（在一起天数、下次纪念日）
- 待办事项管理（创建、完成、删除）
- 快速上传照片入口
- 纪念日提醒卡片
- 近期照片预览

**用户流程**:
1. 未登录用户 → 显示欢迎信息，引导登录
2. 已登录但未加入情侣空间 → 显示创建/加入空间引导
3. 已加入情侣空间 → 显示完整的首页功能

### 2. 时间线 (Timeline)
**目标**: 按时间顺序展示情侣间的所有活动记录

**交互需求**:
- 时间线可视化展示（垂直时间轴）
- 活动分类筛选（全部、待办、照片、纪念日）
- 活动详情查看
- 时间线无限滚动加载
- 按月份分组显示

**活动类型**:
- **待办完成** - 用户完成了某个待办事项
- **照片上传** - 用户上传了新照片
- **纪念日创建** - 用户添加了新纪念日
- **空间创建** - 情侣空间被创建
- **伴侣加入** - 伴侣加入了空间

### 3. 照片 (Photos)
**目标**: 管理情侣共享的照片库

**交互需求**:
- 照片网格布局展示
- 多种筛选方式（全部、我上传的、伴侣上传的、按日期）
- 照片上传功能（支持多选）
- 照片编辑模式（批量选择、删除）
- 照片详情查看
- 照片评论和点赞（未来功能）

**上传流程**:
1. 点击上传按钮 → 选择照片（最多9张）
2. 添加照片描述 → 开始上传
3. 显示上传进度 → 上传完成通知

### 4. 个人资料 (Profile)
**目标**: 管理用户账户和情侣空间设置

**交互需求**:
- 用户个人信息展示
- 情侣空间状态管理
- 邀请码生成和分享
- 统计数据展示
- 应用设置

**标签页结构**:
1. **个人资料** - 用户信息和统计数据
2. **情侣空间** - 空间管理和邀请功能
3. **设置** - 应用设置和帮助

## 情侣空间管理流程

### 创建空间流程
1. 用户点击"创建情侣空间"
2. 确认创建 → 生成8位邀请码
3. 显示邀请码模态框
4. 分享邀请码给伴侣
5. 伴侣输入邀请码加入

### 加入空间流程
1. 用户点击"加入情侣空间"
2. 输入8位邀请码
3. 验证邀请码有效性
4. 成功加入 → 更新情侣状态

### 离开空间流程
1. 用户点击"离开空间"
2. 确认离开（警告数据将丢失）
3. 清除情侣相关数据
4. 返回单身状态

## 用户认证流程

### 登录流程
1. 用户点击"立即登录"
2. 调用微信`getUserProfile` API
3. 获取用户头像和昵称
4. 更新全局登录状态
5. 跳转回原页面

### 退出登录流程
1. 用户点击"退出登录"
2. 确认退出
3. 清除所有用户数据
4. 返回未登录状态

## 数据同步策略

### 实时数据
- 用户登录状态
- 情侣空间状态
- 当前页面数据

### 离线支持
- 待办事项本地缓存
- 照片上传队列
- 网络状态检测

## 错误处理

### 网络错误
- 显示网络错误提示
- 自动重试机制
- 离线模式切换

### 权限错误
- 用户拒绝授权提示
- 引导重新授权
- 功能降级处理

### 数据错误
- 数据加载失败提示
- 本地缓存回退
- 手动刷新选项

## 性能要求

### 加载时间
- 首屏加载 < 2秒
- 页面切换 < 1秒
- 图片加载 < 3秒

### 内存管理
- 图片懒加载
- 列表虚拟滚动
- 数据分页加载
